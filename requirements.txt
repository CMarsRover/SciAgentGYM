# =============================================================================
# SciAgentGYM — requirements.txt (pip-driven install)
# =============================================================================
# Purpose:
#   Full pip dependency list for the SciAgentGYM evaluation harness
#   (agent + 1780+ scientific tools). Modern PyPI wheels cover essentially
#   every scientific package we need.
#
# Version policy:
#   Names only, no version pins. Pip will pick the newest wheel compatible
#   with Python 3.11. This keeps the install script resilient to future
#   yanked / renamed versions and lets reproducers get a "just works"
#   experience out of the box.
#
# NOT included (verl-agent-only, not needed by the evaluation harness):
#   torch / vllm / flash-attn / ray / tensordict / liger-kernel / transformers /
#   accelerate / peft / datasets / wandb / swanlab / xformers / triton / cupy /
#   tensorboard.
#
# Recommended install:
#   conda env create -f environment.yml   # creates env + runs pip install -r
#   conda activate sciagentgym
#
# Alternative (pure pip, no conda; works on modern Linux with Python 3.11):
#   python -m venv .venv && source .venv/bin/activate
#   pip install --upgrade pip setuptools wheel
#   pip install -r requirements.txt
#
# Python: 3.11 recommended.
# =============================================================================
